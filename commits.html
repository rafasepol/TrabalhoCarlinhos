O que é Commit

Definição

Commit é o registro das alterações feitas em um projeto utilizando um sistema de controle de versão, como o Git.

Ele funciona como um ponto de salvamento do código em determinado momento.

Para que serve

  • Salvar alterações no projeto
  • Manter o histórico de mudanças
  • Permitir retornar a versões anteriores
  • Organizar o desenvolvimento em equipe

Como fazer um commit

git add .
git commit -m "Descrição da alteração"

Boas práticas

  • Escrever mensagens claras
  • Fazer commits pequenos e organizados
  • Evitar mensagens genéricas
