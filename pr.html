O que é um Pull Request?

Definição

Um Pull Request (PR) é um método de submeter contribuições para um projeto de software. É a forma como você avisa aos outros que as alterações feitas em uma branch estão prontas para serem revisadas e mescladas (merged) no código principal.

Como funciona o fluxo?

  • Criação: Você cria uma branch, faz alterações e as envia para o repositório remoto.
  • Discussão: A equipe revisa o código, faz comentários e sugere melhorias diretamente no PR.
  • Testes: Ferramentas automáticas verificam se o novo código quebra algo existente.
  • Merge: Após a aprovação, o código é integrado ao projeto principal.

Por que usar?

PRs promovem a colaboração, garantem a qualidade do código através da revisão por pares e mantêm um histórico transparente de quem mudou o quê e por quê.
