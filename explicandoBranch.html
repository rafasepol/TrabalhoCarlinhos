O que é uma Branch?

Conceito

Uma branch é basicamente uma linha paralela de desenvolvimento dentro de um projeto. Ela permite que você trabalhe em novas funcionalidades ou correções sem mexer diretamente na versão principal.

Por que usar?

  • Evita quebrar o código principal
  • Permite testar novas ideias
  • Facilita o trabalho em equipe

Exemplo no Git

Criar uma nova branch:

git branch minha-feature

Trocar para ela:

git checkout minha-feature

Ou tudo em um comando:

git checkout -b minha-feature
